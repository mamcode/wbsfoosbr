Odoo Multi-Website

One single website isn't enough to satisfy your requirement?
What will you do if all your requirements cannot be fulfilled by a single website? Here we have the solution to this problem. Odoo Multi-Website will help you to set-up multiple websites with Multi-Theme, with this module you can create Multi-Websites with same or different Company, handle multiple portals, manage separate Menus, Icons, Favicons and many other features. Now increase traffic to your websites, boost SEO manage the content breakup and all this with a single database.

Salient features
  • Manage Multi-Websites without any limitation with a simple click from a single database.
  • Different domains for different Websites by simply adding the domain.
  • Multi-Websites with same or different Company.
  • Multi-Websites with different Website Themes.
  • Multi-eCommerce based on categories.
  • Different Menus for Multiple Websites.
  • Different price-lists for different Websites.
  • Different Shipping Methods for different Websites.
  • Different Payment Methods for different Websites.
  • Different Categories for different Websites.
  • Different Logos and Favicons for different Websites.
  • Different Social Media links to different Websites.
  • Filter Reports on the basis of Websites.
  • Filter Orders, Invoices, Pickings, Customers on the basis of Websites.
  • Filter Website Sale analytics on the basis of Websites.
  • Website Dashboard data changes on Changing the Website.
  • Go to different Websites directly from admin account by a simple click.

Backend of Multi Websites with filters.

Check Sales Reports on the basis of website.

Dashboard will be displayed for the selected site. On clicking the Go To Website it will be redirected to the selected site. Sale analytics of the selected Website will be displayed.

Filter Customers on the basis of website. Keep the record of customers from all Websites on which he logged in.

Filter Orders on the basis of website. Check website for a particular Order.

Filter Pickings on the basis of website. Check website for a particular Picking.

Filter Invoicing on the basis of website. Check website for a particular Invoicing.

Configure Multi Websites.

Create A Website and set some values like

Website Name, Logo, Favicon, Domain, Languages, Category, Theme, Company, Delivery Methods, Price-lists, Menus, Social Media Links etc.
